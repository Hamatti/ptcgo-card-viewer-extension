Visual Testing Page for PTCGO Card Viewer

How to use?

To use this page for testing, run make visual-text in the project root. This will run a local fileserver on localhost:9000, open up a Firefox browser to localhost:9000 and load the extension.

Run the extension by clicking at the icon on toolbar. Verify the following:

  • That each item in the PTCGO Code column is highlighted
  • That hovering over each item displays an image
  • That the image matches the image in Card image column
